SDG (SmartDataGrid)

SmartDataGrid is a sophisticated fully featured jQuery based data grid control.

Features

✔ Dynamic data loading
✔ Virtual scrolling
✔ Dynamic data grouping
✔ Data filtering
✔ Column sorting, re-ordering, re-sizing
✔ State persistance
✔ Easy to skin/theme (purely CSS driven)
✔ Support for custom filters, cell renderers, sorting
✔ Tablet support (iOS/Android)
✔ Support for mock to enable faster development
✔ JSON based interface

Browser support

✔ IE8+
✔ FireFox
✔ Chrome
✔ Safari
✔ Opera
✔ iOS Safari
✔ Android Chrome

Examples

  1. Basic
  2. With filtering, grouping, sorting, state persistance and other features
  3. With custom renderer and sorting
  4. With custom filter
  5. With custom plug-in
  6. Constructor properties
  7. Column properties
  8. Real-time updates

Note: All the examples are hosted on JSFiddle, so you can change and play with it whenever and however you want.
